# E-Commerce Sales Dashboard

This project analyzes e-commerce sales data using Power BI and Excel to uncover sales trends, profit patterns, and top-performing products.

## Tools Used
- Power BI
- MS Excel

## Key Insights
- Monthly and quarterly sales trends
- State-wise performance analysis
- Top-selling products and profit analysis
- Interactive KPIs and filters
